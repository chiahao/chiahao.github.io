title: "Response.entity to Prevent HTML Page in Jersey"
date: 2024-04-15 11:32:00
category: jersey3
tags: [jersey3]
---

use `.entity()` to inhibit default returned HTML.

```java
return Response
	.status(UNAUTHORIZED)
	.entity("Custom Message")
	.type(MediaType.APPLICATION_JSON)
	.build();
```
